# grant_v08 — Grant Tracker (scaffold)

Minimal full-stack scaffold for the Grant Tracker app (initial skeleton).

Structure:
- `server/` — Express server that serves client and a simple API.
- `client/` — Static client with floating scorecard UI.

Run locally:

1. Install dependencies:

```bash
npm ci
```

2. Start server:

```bash
npm start
```

The server listens on port 3000 by default and serves the client UI.

Notes:
- This is an initial scaffold for review and CI setup. Tests, CI, and production readiness will be added next.
